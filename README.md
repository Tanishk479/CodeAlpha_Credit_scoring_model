# CodeAlpha_Credit_scoring_model
# Creditworthiness Prediction Using Machine Learning

## Overview
This project predicts whether a customer is creditworthy or not using machine learning algorithms. The system analyzes financial information such as income, loan amount, and credit history to classify customers into safe or risky categories.

The project uses classification algorithms including:
- Logistic Regression
- Decision Tree
- Random Forest

---

## Objective
The main objective of this project is to build a machine learning model that can predict the creditworthiness of individuals based on their previous financial records.

---

## Dataset
Dataset used:
Loan Prediction Dataset from Kaggle

Dataset Link:
https://www.kaggle.com/datasets/altruistdelhite04/loan-prediction-problem-dataset

---

## Features Used
The following features were used in the project:

- Applicant Income
- Loan Amount
- Credit History
- Gender
- Education
- Property Area
- Marital Status
- Dependents

Feature engineering was also applied using:
- Debt Income Ratio
